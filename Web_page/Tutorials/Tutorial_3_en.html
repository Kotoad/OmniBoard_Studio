Goal is to demonstrate different LED patterns and timings.

You will need:

  • Any Raspberry Pi board
  • USB cable to connect the board to your computer
  • 3x Any LED (different colors recommended)
  • 3x 220 Ohm resistor
  • Breadboard and jumper wires

(Wiring diagram image would go here)

How to make it work:

  1. Create new varibles by clicking the "Variables" button in the main window toolbar.
    Name them however u want and set their type to Boolean.
  2. Create new devices by clicking the "Devices" button in the main window toolbar.
    Name them LED1, LED2, and LED3 for convenience and set their type to Output.
  4. Add the required elements to the canvas as described in the previous examples.
  5. Connect the elements to create different LED patterns with varying timings.
